# erne-universal

Complete AI coding agent harness for React Native and Expo development.

## Overview

**erne-universal** (Everything React Native & Expo) provides a comprehensive suite of AI-powered agents, commands, skills, and MCP integrations purpose-built for React Native and Expo development workflows.

## What's Included

| Component | Count | Description |
|-----------|-------|-------------|
| **Agents** | 8 | Specialized AI experts for RN/Expo development |
| **Commands** | 16 | Slash commands for common mobile dev tasks |
| **Skills** | 8 | Reusable capabilities for mobile workflows |
| **MCP Integrations** | 10 | Tool and service connections |

## Categories

- `mobile-development` — Native mobile app development workflows
- `react-native` — React Native specific tooling and patterns
- `ai-agents` — AI-assisted development agents

## Installation

```bash
npx erne-universal init
```
