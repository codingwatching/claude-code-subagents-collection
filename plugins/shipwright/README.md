# Shipwright

Describe your app in plain English — Shipwright builds, tests, and deploys it autonomously. 9-phase pipeline, 4 stacks, enterprise-grade safety hooks.

## Installation

```bash
/plugin install shipwright@buildwithclaude
```

## What's Included

- **Agent**: `shipwright` — Autonomous app-building agent
- **Commands**: `/shipwright:build`, `/shipwright:enhance`, `/shipwright:stacks`, `/shipwright:projects`
- **Skills**: `build` — Full pipeline execution

## Supported Stacks

- Next.js + TypeScript + Tailwind
- FastAPI + Python
- Express + TypeScript
- Static HTML/CSS/JS
