## Color Psychology

### Color Meanings

| Color | Associations | Best For |
|-------|--------------|----------|
| **Black** | Luxury, power, sophistication | Premium brands, tech, fashion |
| **White** | Clean, minimal, pure | Healthcare, minimal brands |
| **Red** | Energy, urgency, passion | Food, entertainment, sales |
| **Orange** | Friendly, energetic, creative | Startups, creative, food |
| **Yellow** | Optimism, warmth, caution | Children's, food, warnings |
| **Green** | Nature, growth, money | Finance, eco, health |
| **Blue** | Trust, calm, professional | Corporate, tech, finance |
| **Purple** | Luxury, creativity, wisdom | Beauty, spiritual, creative |
| **Pink** | Playful, feminine, modern | Fashion, beauty, youth |
| **Brown** | Earthy, reliable, warm | Food, outdoor, organic |

### Warm vs Cool

**Warm colors** (red, orange, yellow):
- Feel energetic and inviting
- Come forward visually
- Best for CTAs and accents

**Cool colors** (blue, green, purple):
- Feel calm and professional
- Recede visually
- Best for backgrounds and large areas

## Creating Custom Palettes

### From a Single Color

1. Start with primary brand color
2. Generate shades (lighter/darker)
3. Choose complementary accent
4. Define text colors with proper contrast

### Color Harmony Rules

**Monochromatic**: Single hue, varied lightness
- Safe, cohesive, can lack energy

**Complementary**: Opposite on color wheel
- High contrast, vibrant, use sparingly

**Analogous**: Adjacent on color wheel
- Harmonious, natural, low contrast

**Triadic**: Three evenly spaced colors
- Balanced, vibrant, complex to execute

### Shade Generation

From a base color, create a scale:

```
50:  Very light (background tint)
100: Light
200: Light medium
300: Medium light
400: Medium
500: Base color
600: Medium dark
700: Dark medium
800: Dark
900: Very dark (text)
950: Near black
```

## Accessibility Reference

### Minimum Contrast Ratios

| Content Type | Minimum Ratio | Level |
|--------------|---------------|-------|
| Normal text | 4.5:1 | AA |
| Large text (18px+) | 3:1 | AA |
| UI components | 3:1 | AA |
| Enhanced text | 7:1 | AAA |

### Safe Text Colors

**On white backgrounds:**
- `#171717` (contrast: 16:1)
- `#404040` (contrast: 10:1)
- `#525252` (contrast: 7:1)
- `#737373` (contrast: 4.5:1) - minimum

**On black backgrounds:**
- `#ffffff` (contrast: 21:1)
- `#e5e5e5` (contrast: 16:1)
- `#a3a3a3` (contrast: 7:1)
- `#8a8a8a` (contrast: 4.5:1) - minimum
